# absMoreModelnames

Fork of [absTXDNames](https://github.com/ins1x/moonloader-scripts/tree/main/absolute-play/absTXDNames)

Описание: Добавляет названия для некоторых объектов в редакторе карт в Absolute Play  
Активация: автоматическая  

> Дополнительно добавляет возможность переключения страниц на стрелки и боковые клавиши мыши

В меню создания объектов теперь все эффекты подписаны:

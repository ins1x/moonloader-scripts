# mini-calc
Простой калькулятор с минимальным набором функций

![](https://github.com/ins1x/moonloader-scripts/raw/main/mini-calc/demo.png)  

**Активация: `/calc`**

### Возможности

* Есть поддержка хоткеев с numpad и расширенной клавиатуры
* Поддержка основных математических операций
* Отображает последнюю операцию
* Минималистичный интерфейс на imgui

> В этой версии не работают операции с дробями (float)

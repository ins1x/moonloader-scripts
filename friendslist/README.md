# FriendsList

![](https://github.com/ins1x/moonloader-scripts/blob/main/friendslist/demo.png?raw=true)

Минималистичный скрипт который отображает кто из твоих друзей в сети, и уведомляет о их входе либо выходе из игры. Эта версия является доработкой  скрипта [SA:MP Player Notifer](https://www.blast.hk/threads/66784/) от автора Nisad Rahman

> Активация: Автоматическая

### Команды:

* /friends add Никнейм/ID - добавить в список друзей
* /friends del Никнейм/ID - удалить из списка друзей
* /friends list - список друзей
* /friends help - помощь
